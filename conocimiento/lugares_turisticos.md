# Lugares Turísticos de Quezada

Aunque es un municipio tranquilo, Quezada tiene varios puntos de interés.

## Parque Central
El corazón del pueblo, un lugar ideal para relajarse por la tarde. Recientemente remodelado.

## Cerro "El Mirador"
Para los más aventureros, se puede subir a este cerro para tener una vista panorámica de todo el valle.
